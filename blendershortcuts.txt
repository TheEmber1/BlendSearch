Viewport Navigation
Middle Mouse Button (MMB): Rotate view
Shift + MMB: Pan view
Ctrl + MMB or Scroll Wheel: Zoom view
Numpad 0: Camera view
Numpad 1: Front view
Numpad 3: Right view
Numpad 7: Top view
Numpad 5: Toggle perspective/orthographic view
Numpad 2/4/6/8: Rotate view
Numpad 9: Opposite view
Home: Frame all
Period (.): Frame selected
Slash (/) or Numpad /: Isolate selection (Local view)
Shift + Z: Toggle render preview
Alt + MMB: Orbit around selection
Shift + F: Walk/Fly mode
Ctrl + Alt + Q: Toggle quad view
Shift + B: Zoom to box
Alt + Home: View all objects in all viewports

K: Set keyframe menu
